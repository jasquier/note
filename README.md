# Notes

A note taking application.

---

## Usage

Display help:

`note [--help]`

Create general note:

`note 'work on your projects'`

Create note under a topic:

`note -t garden 'weed the strawberries'`

List all notes:

`note --list`

List all notes under a topic:

`note --list -t <topic>`

List all topics:

`note --list -t`
